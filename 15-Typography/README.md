# Typography Basics

Typography is the heart of a website. Proper stylings on text, makes website look better, productive and standard.

## Available CSS Properties (cssreference.io)
 - color
 - font-family
 - font-size
 - font-style
 - font-variant
 - font-weight
 - font
 - letter-spacing
 - line-height
 - text-align
 - text-decoration
 - text-indent
 - text-overflow
 - text-shadow
 - text-transform
 - white-space
 - word-break
 - word-spacing

See these properties in action [here](https://cssreference.io/typography/).
